# 01 リクルータースクリーニング対策

**日程：** 2026年6月30日（月）
**時間：** 招待状は20分。ただし実際は30〜45分になる可能性あり（後述）
**言語：** 日本語（英語の可能性もあり）

---

## ⚠️ 所要時間について（Web調査による修正）

招待状に記載の20分は通常より短い。複数のソースによると：

| ソース | FDEリクルータースクリーンの所要時間 |
|---|---|
| Exponent（FDE専門ガイド） | **30〜45分** |
| Glassdoor 実体験レポート | **45分** |
| Google一般採用プロセス | 30〜60分 |

**対応方針：**
- 招待の20分はあくまで最短の目安として受け取る
- **30分分の内容を準備**し、最初の20分をタイトに構成する
- 延長しても慌てない（むしろ会話が盛り上がっているサイン）
- 「もう少し時間はありますか？」と聞かれたら快諾する

---

## このラウンドで見られること

| 評価軸 | 内容 |
|---|---|
| 志望動機 | なぜFDEか、なぜGoogle Cloudか |
| 経験の関連性 | 本番AIシステムの構築・デプロイ経験があるか |
| コミュニケーション | 技術的な仕事を非技術者にも伝えられるか |
| 経験の幅 | エージェント/MLと古典的エンジニアリング両方を語れるか |
| 逆質問の質 | ロールへの理解度と準備度 |

## 想定タイムライン

### パターンA：20分で終わる場合（最短シナリオ）

```
00:00 - 02:00  自己紹介（促された場合）
02:00 - 08:00  志望動機・バックグラウンドの確認
08:00 - 15:00  プロジェクト深掘り（1件）
15:00 - 18:00  ロジスティクス確認（年収・入社可能時期）
18:00 - 20:00  逆質問（1問に絞る）
```

### パターンB：30〜45分に伸びる場合（想定すべき標準シナリオ）

```
00:00 - 03:00  自己紹介・アイスブレイク
03:00 - 10:00  志望動機・バックグラウンドの確認
10:00 - 22:00  プロジェクト深掘り（1〜2件）
22:00 - 32:00  技術的な会話・詳細確認
32:00 - 38:00  ロジスティクス確認（年収・入社可能時期）
38:00 - 45:00  逆質問（2問）
```

**回答の目安：** 各回答は1〜2分以内。深掘りされたら3分まで。
